# Classmaster

## Development

### Initial setup

  Setup database and start server:

  ```
    rails db:setup
    rails s
  ```

  Specs:

  ```
    rails test
  ```
  Run specific test:

  ```
    ruby -I test  test/integration/clubs_login_test.rb
  ```

### Dependencies

  Ruby 2.6.5
  ostgreSQL 11.4
